# Customer Churn Prediction using Machine Learning

## Overview
This project predicts telecom customer churn using Exploratory Data Analysis (EDA) and Machine Learning. The objective is to identify customers likely to leave and provide business recommendations to improve retention.

## Dataset
- Telco Customer Churn Dataset
- ~7,000 customer records
- Target: `Churn`

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Google Colab

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Correlation Analysis
5. Logistic Regression
6. Random Forest
7. Model Evaluation
8. ROC Curve
9. Feature Importance
10. Business Recommendations

## Model Performance

| Model | Accuracy | Precision | Recall | F1-Score | ROC-AUC |
|-------|---------:|----------:|--------:|---------:|--------:|
| Logistic Regression | 80.38% | 64.76% | 57.49% | 60.91% | 0.836 |
| Random Forest | 79.03% | 63.30% | 50.27% | 56.04% | 0.817 |

## Key Insights
- Long-term customers are less likely to churn.
- Month-to-month contracts have the highest churn.
- Fiber optic customers churn more frequently.
- Higher monthly charges increase churn risk.
- Electronic check users have higher churn.

## Business Recommendations
- Promote long-term contracts.
- Improve fiber optic service quality.
- Encourage automatic payment methods.
- Launch retention campaigns for new customers.
- Offer loyalty rewards and personalized discounts.
