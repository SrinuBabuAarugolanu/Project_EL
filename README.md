## 2️⃣ Customer Churn Prediction Using Machine Learning

### Overview

Developed a machine learning model to predict telecom customer churn using customer demographics, subscription details, and billing information.

### Project Workflow

- Data Cleaning
- Exploratory Data Analysis
- Feature Engineering
- Correlation Analysis
- Logistic Regression
- Random Forest
- ROC Curve Analysis
- Feature Importance
- Business Recommendations
- Power BI Dashboard

### Machine Learning Results

| Model | Accuracy | ROC-AUC |
|--------|----------|---------|
| Logistic Regression | **80.38%** | **0.836** |
| Random Forest | **79.03%** | **0.817** |

### Key Insights

- Month-to-month contracts have the highest churn.
- Fiber optic customers churn more frequently.
- Higher monthly charges increase churn risk.
- Electronic check users have higher churn.
- Long-term customers are more loyal.

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
